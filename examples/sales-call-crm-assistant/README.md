# Sales call CRM assistant

Synthetic case study showing a medium-risk structured extraction workflow for sales reps updating CRM fields from call transcripts.

## Recommendation

**Pilot ready with conditions.** The product can enter a small pilot if consent is manually verified, observability is live, reps are trained, and the eval set is completed during the pilot.

## Files

- `opportunity-brief.md`: why CRM update automation is worth testing
- `prd.md`: AI job, transcript inputs, Salesforce output contract, review rules
- `eval-plan.md`: extraction accuracy, evidence checks, consent and quality cases
- `launch-gate.md`: readiness assessment and recommendation
- `readiness-assessment.yaml`: structured scoring input

Names, dates, metrics, and reviewers are illustrative.
